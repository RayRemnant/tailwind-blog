Navigate to the folder you want to download on GitHub website.

Copy the URL, it shoud look like this: `https://github.com/REPO_OWNER/REPO_NAME/tree/main/PATH_TO_FOLDER`

Go to https://download-directory.github.io/ and paste the URL.

Press Enter and your download will immediately start!
