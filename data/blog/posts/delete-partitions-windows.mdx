Delete partitions that cannot be deleted using the Disk Management on Windows

Open terminal, type `diskpart`

`list disk`

You will see a list

`select disk 0`

Change the number to match your desired disk.

`list partition`

You will see a list of partitions.

`select partition 0`

Change the number to match your desired partition.

`delete partition override`

This will delete the partition.
